# SCMK 参数敏感性分析

对 SCMK(scatter) 的三个超参数做敏感性分析，在 **20 个论文数据集**上、对 **seed ∈ {0,1,2}**
求均值（误差带 = 跨数据集 std）。

| 参数 | 含义 | 取值范围 | 数据来源 |
|---|---|---|---|
| `d` | 隐藏层（嵌入）维度 | 16, 32, 64, 128, 256 | `hybrid_semi_all.csv` |
| `λ` | scatter 损失权重 | 0, 0.1, 1, 10, 100, 1000 | `hybrid_semi_all.csv` |
| `ν` | OC-SVM 决策边界松紧 | 0.01–0.5 | 本目录补跑（只动 OC-SVM 评分阶段）|

边际曲线口径：固定该参数为某值时，每数据集在**其余参数上取最优**、再对 seed 平均，最后对数据集求均值。

## 主要结论

- **`λ`（scatter 权重）= 最敏感**，极差 **0.165**。曲线非单调：
  λ=0(无 scatter) 0.797 → λ=0.1/1 反而更低(0.725/0.742) → λ≥10 显著抬升并在 **100** 达峰(0.890)，1000 仍稳。
  说明：**弱 scatter 反而干扰跨核一致性**，必须给到 λ≥10 才能换来紧致性收益——印证 scatter 损失是核心组件，推荐 λ∈[10,1000]。
- **`d`（隐藏维度）= 鲁棒**，极差仅 **0.032**，d=32~256 基本持平，**128** 略优(0.890)。
- **`ν`（OC-SVM）= 鲁棒**，极差 **0.038**，0.01–0.2 几乎持平、偏小更好，ν>0.3 才缓慢下降。
  （此处对两路 OC-SVM 用同一 ν；论文主结果允许两路各自调 ν，故绝对值略高于本图。）

**一句话**：SCMK 对 d 和 ν 不敏感（开箱即用），唯一需要保证的是 λ 取得足够大（≥10）。
